留意事項
	YukkuriMovieMakerは、饅頭遣い様の作品です。
	このテキストに書いてあることは、あくまでMaterial_Finder(以下、本ソフト)に関する各ファイルに不具合がないと仮定した場合の挙動です。ご了承ください。

ymmp.dllには実際に走査・出力する機能はありません。
代わりにymmp.dllは以下のうち、あなたが指定したDLLをさらに呼び出しそれに走査・出力を任せます。
指定するタイミングは、
・ymmp.dllの初回ロード(初回呼び出し)時
・フォーマット欄に特定のコマンドを打った場合
のいずれかです。
以下は、ymmp.dllが呼び出し得る.ymmpファイル専用DLLの仕様等です。


・YMM_4.29.0.2.dll
	対応するYMMのバージョン
		v4.29.0.2
		(v4.0.0.0前後まで対応している可能性がありますが、正式な動作確認はしていません。)

	素材を出力する基準
		・非表示でない
		・プレビュー時のみ表示エフェクトが直接かかってない
		・遅延エフェクトの拡大率が0より大
		・単色化で設定されている色の不透明度が0より大
		・合成音声/フォントの場合、テキスト/字幕がある
		・音声の場合、音量と再生速度がどちらも0より大
		など...
		YMM4の仕様上、見える・聞こえるはずがないものは出力されないことになっています。ただし、その判断がプログラム上難しいものは出力されます。よって、
		出力されなかったものは全て動画でも見えない・聞こえないはずですが、出力されたものすべてが動画上で見える・聞こえるというわけではありません。
		たとえば、不透明度エフェクトは出力するかどうかの判断に利用されないため、不透明度0%の素材は他の基準に引っかからなければ必ず出力されます。
		詳しく知りたい場合は開発者までご連絡ください。

	接頭辞/ファイル形式入力欄
		接頭辞
			例えば「C:」と入力すれば、出力基準を満たした素材のうちCドライブにあるものだけを出力します。
		フォーマット形式
			例えば「mp3」と入力すれば、出力基準を満たした素材のうちmp3ファイルだけを出力します。
		この2つは併用できます。
		また、絶対パスの1文字目ではなく、ファイル名の1文字目を基準として接頭辞を設定したい場合は、ファイル形式欄に「*ONLYFILENAME」コマンドを入力することで可能です。
		例えば、ncから始まるファイル名の素材を全て出力するには、接頭辞に「nc」, ファイル形式に「*ONLYFILENAME」と入力してください。
		詳しくは、以下をご覧ください。
		
	使用できるコマンド
		コマンド名 || 動作 ([]は、空欄でも　指定してもよい、という意味です)

		ファイル形式欄(右側)に入力できるもの
			*DLLSELECT
				走査対象ファイルが変わるたびにDLL選択画面を表示します(注 : 「DLLSELECT」ではなく*DLLSELECTです。以下も同じ)。出力時は無視されます。
			*DLLSELECT1
				ymmp.dllがロードされる毎にDLL選択画面を表示します。
				例えば、走査対象ファイルをa.ymmp, b.ymmp, c.txt, d.ymmp, e.ymmpの順で選択した場合、a.ymmpとd.ymmpの走査前の2回だけ表示されます。出力時は無視されます。
			*ALL / *全て
				出力基準を満たした素材を全て出力します。
			*AUDIO / *音声 / *音源
				出力基準を満たした音声素材を全て出力します。[図形アイテム:波形]で使われる音声ファイルも含まれます。
			*AUDIO- / *音声ー / ＊音源ー
				出力基準を満たした音声素材を全て出力します。[図形アイテム:波形]で使われる音声ファイルは含まれません。
			*VIDEO / *動画
				出力基準を満たした動画素材を全て出力します。
			*IMAGE / *画像
				出力基準を満たした画像素材を全て出力します。SVGファイルとディスプレイメントマップとして使用されたファイルも含まれます。
			*FONT / *フォント
				出力基準を満たしたフォントを全て出力します。
			*TACHIE / *立ち絵
				出力基準を満たした立ち絵素材(=シンプル立ち絵 + 動く立ち絵 + PSD立ち絵 + その他の全ての立ち絵)を全て出力します。
			*TACHIE_SIMPLE / *シンプル立ち絵
				出力基準を満たしたシンプル立ち絵素材を全て出力します。
			*TACHIE_ANIMATION / *動く立ち絵
				出力基準を満たした動く立ち絵素材を全て出力します。
			*TACHIE_PSD / *PSD立ち絵
				出力基準を満たしたPSD立ち絵素材を全て出力します。
			*TACHIE_PLUGIN / *プラグイン立ち絵
				出力基準を満たした立ち絵素材のうち、プラグインを経由して使用された立ち絵を全て出力します。
			*VOICEAPI / *合成音声
				出力基準を満たした合成音声素材を全て出力します。
			*PLUGIN /*プラグイン
				出力基準を満たしたプラグインを全て出力します。
			*ONLYFILENAME / *ディレクトリなし / *ディレクトリ名なし
				接頭辞欄に何も入力しなかった場合 : 出力基準を満たした素材を全てファイル名だけ出力します。
				接頭辞欄に何かしら入力した場合　 : ファイル名がその接頭辞から始まる素材を出力します。
					例えば、接頭辞に「nc」, ファイル形式に「*ONLYFILENAME」と入力すればncから始まる名前の素材が出力されます。これはフォント名などにも利用可能です。

		接頭辞欄(左側)に入力できるもの
			*ONLYFILENAME / *ディレクトリなし / *ディレクトリ名なし
				出力基準を満たした素材を全てファイル名だけ出力します。ファイル形式欄に入力できるコマンドとのみ併用できます。
				ファイル形式欄にmp3などと入力しても反映されません。(その場合、*ONLYFILENAMEは要らないからです。)ご注意ください。


	v1と同様、当然接頭辞とフォーマットを指定する方法も利用できます。ただし、このDLLではファイル形式の前に「.」があるかどうかを判断しません。
	これは、このDLLはフォント名なども素材として出力するため、汎用性を考えてそのように設定されました。

	制限
		最大識別キャラクター数
		2000人(2001人目以降のキャラクターの使用合成音声を素材として出力することはできない、という意味です。)
		出力の詳細設定の設定は、出力ウィンドウの更新頻度以外全く反映されません。
		読み込める素材数には上限はありません。
