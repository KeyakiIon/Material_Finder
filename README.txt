本体ver3.0.0

留意事項
	本バージョンは以前のバージョンと互換性がありません。
	LICENSEも短いので必ずお読みください。

新機能の使い方
	ファイルのドラッグ&ドロップ
		走査するファイルは、起動時又は起動中にドラッグ&ドロップすることで設定可能です。
		出力するファイルは、起動中、Shiftを押しながらドロップすることで設定可能です。
	

お詫び
	開発の長期化により、本体のデバッグが終わっていない状態でのリリースとなりました。（特にショートカットキー）
	ご不便をおかけして申し訳御座いません。
	もしよろしければ、動いた・動かなかったなどだけでも構いませんので、少しでもご利用報告いただけますと大変ありがたく思います。

注意事項
	ファイルやソフトを別のディレクトリに移動すると、設定データを読み込めなくなります。
	

質問
	・v1.0.0の時にあったdefault.dllがありませんが？
		YMMへの対応を急いだため、本体のバージョンアップにdefault.dllを追いつかせることができませんでした。
		申しわけありません。
		ver1台のdefault.dllはver2以降に対応していませんので、DLLディレクトリに配置することはおやめください(クラッシュしかねません)。
	
ヘルプ
	・出力中、強制的にソフトを終了させる方法
		E, N, Dキーを3つとも押しながらメインウィンドウの✕ボタンをクリック
